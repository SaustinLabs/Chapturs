# Screenshots

Drop screenshots here and they will automatically appear in the **"See It In Action"** section on `/features`.

The section only renders for files that are actually present — no code changes needed to add or remove a screenshot.

## Expected filenames

| File | What to screenshot |
|------|--------------------|
| `reader.png` | A chapter page mid-scroll, ideally with a glossary tooltip open |
| `glossary.png` | The glossary panel / a term tooltip |
| `featured-comments.png` | The story front page with featured comments visible |
| `creator-profile.png` | A creator profile page with blocks |
| `analytics.png` | The analytics dashboard in the Creator Hub |
| `rating.png` | The multi-dimension rating modal |

## Tips

- **16:10 aspect ratio** works best (e.g. 1280×800 or 1600×1000)
- PNG is fine; WebP is also supported if you rename the entries in `features/page.tsx`
- Screenshots are served at `/screenshots/<filename>` from this folder
